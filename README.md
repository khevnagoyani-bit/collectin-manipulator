# collectin-manipulator

🎓 Student Data Organizer

«A smart command-line application for managing student records efficiently using Python.»

Student Data Organizer is a menu-driven Python application that simplifies student record management. It allows users to add, view, update, and delete student information while maintaining a unique collection of subjects offered. The project is designed to demonstrate how different Python data structures can work together to solve real-world data management problems.

---

🌟 Overview

Managing student information is a common requirement in schools, colleges, and training centers. This application provides a simple yet effective way to organize student records directly from the command line.

Each student record stores personal information, academic details, and enrolled subjects. The application also maintains a unique list of all subjects available using Python's "set" data structure.

---

✨ Key Features

- 👨‍🎓 Add new student records
- 📋 Display all student information
- ✏️ Update a student's age and subjects
- 🗑️ Delete student records
- 📚 Display all unique subjects offered
- 🆔 Store Student ID and Date of Birth as a tuple
- 📂 Organize records using dictionaries and lists
- ⚡ Interactive menu-driven interface
